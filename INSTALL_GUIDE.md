# 在 iPhone 上安装 Time Capsule 应用

应用已配置为 PWA (Progressive Web App)，可以在 iPhone 上安装到主屏幕，就像原生应用一样！

## 📱 安装步骤

### 方法一：通过 Safari 浏览器安装（推荐）

1. **确保设备在同一网络**
   - iPhone 和电脑连接到同一个 Wi-Fi 网络

2. **在 iPhone Safari 中打开应用**
   - 打开 Safari 浏览器
   - 输入地址：`http://192.168.1.38:5173`
   - （如果无法访问，尝试 `http://100.64.164.2:5173`）

3. **添加到主屏幕**
   - 点击 Safari 底部的"分享"按钮（方框带向上箭头）
   - 向下滚动，找到并点击"添加到主屏幕"
   - 可以修改应用名称（默认是 "Time Capsule"）
   - 点击"添加"

4. **完成！**
   - 应用图标会出现在主屏幕上
   - 点击图标即可打开应用，就像原生应用一样

### 方法二：通过其他浏览器

- **Chrome**: 点击菜单 → "添加到主屏幕"
- **Edge**: 点击菜单 → "添加到主屏幕"

## ✨ PWA 功能特性

安装后，应用将具备以下特性：

- ✅ **独立运行**：点击图标后以独立窗口打开，没有浏览器地址栏
- ✅ **离线支持**：部分功能可以在离线状态下使用
- ✅ **自动更新**：应用会自动检查并更新到最新版本
- ✅ **原生体验**：全屏显示，体验接近原生应用

## 🔧 故障排除

### 无法访问应用

1. **检查网络连接**
   - 确保 iPhone 和电脑在同一 Wi-Fi 网络
   - 尝试 ping 电脑的 IP 地址

2. **检查防火墙**
   - Windows 防火墙可能阻止了端口 5173
   - 需要在防火墙中允许端口 5173

3. **尝试其他 IP 地址**
   - 如果 `192.168.1.38` 不可用，尝试 `100.64.164.2`

### 无法添加到主屏幕

1. **确保使用 Safari**
   - iOS 上只有 Safari 支持完整的 PWA 功能
   - 其他浏览器可能功能受限

2. **检查 HTTPS**
   - 某些 PWA 功能需要 HTTPS（生产环境）
   - 开发环境使用 HTTP 也可以，但某些功能可能受限

3. **清除缓存**
   - 设置 → Safari → 清除历史记录和网站数据
   - 重新打开应用

## 📝 注意事项

- 开发服务器需要保持运行才能访问应用
- 生产环境建议部署到支持 HTTPS 的服务器
- 首次安装可能需要几秒钟来下载资源

## 🚀 生产部署

如果要部署到生产环境：

1. 构建应用：
   ```bash
   npm run build
   ```

2. 部署 `dist` 目录到支持 HTTPS 的服务器

3. 在 iPhone 上访问 HTTPS 地址并安装
