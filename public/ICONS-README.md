# 图标文件说明

## 需要的图标文件

应用需要以下图标文件，请将它们放在 `public/` 文件夹中：

### PWA 图标（Web 应用）

- `icon-192.png` - 192x192 像素
- `icon-512.png` - 512x512 像素

### iOS 应用图标

iOS 应用需要多种尺寸的图标。可以使用以下工具生成：

1. **在线工具**：
   - https://www.appicon.co/
   - https://www.favicon-generator.org/
   - https://icon.kitchen/

2. **本地工具**：
   - 打开 `icon-generator.html` 在浏览器中生成基础图标
   - 然后使用在线工具生成所有 iOS 尺寸

## iOS 图标尺寸要求

iOS 应用图标需要以下尺寸（@1x, @2x, @3x）：

- 20x20 (@2x, @3x) = 40x40, 60x60
- 29x29 (@2x, @3x) = 58x58, 87x87
- 40x40 (@2x, @3x) = 80x80, 120x120
- 60x60 (@2x, @3x) = 120x120, 180x180
- 76x76 (@1x, @2x) = 76x76, 152x152
- 83.5x83.5 (@2x) = 167x167
- 1024x1024 (@1x) = 1024x1024

## 快速生成图标

### 方法 1: 使用生成器

1. 打开 `icon-generator.html`
2. 下载生成的图标
3. 使用在线工具（如 appicon.co）生成所有 iOS 尺寸

### 方法 2: 使用命令行工具

```bash
# 安装 ImageMagick (如果未安装)
# macOS: brew install imagemagick
# Linux: sudo apt-get install imagemagick

# 从 1024x1024 主图标生成所有尺寸
# (需要先有一个 1024x1024 的图标文件)
```

### 方法 3: 使用设计工具

使用 Figma、Sketch 或 Photoshop 创建图标，然后导出所有需要的尺寸。

## 图标设计建议

1. **简洁明了**：图标应该在小尺寸下也能清晰识别
2. **避免文字**：小尺寸下文字难以阅读
3. **使用品牌色**：Time Capsule 使用紫色渐变主题
4. **圆角处理**：iOS 会自动添加圆角，设计时考虑这一点
5. **背景透明**：使用 PNG 格式，支持透明背景

## 当前图标状态

如果图标文件不存在，应用仍可以运行，但：
- PWA 安装时可能没有图标
- iOS 应用会使用默认图标

建议在打包前准备好所有图标文件。
