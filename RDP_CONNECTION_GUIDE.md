## 🖥️ 方法二：手动配置 Remote Desktop

### 步骤 1：打开 Remote Desktop

**Windows 10/11:**
- 按 `Win + R`
- 输入 `mstsc`
- 按回车

**或搜索:**
- 按 `Win` 键
- 输入 "Remote Desktop"
- 打开"远程桌面连接"

### 步骤 2：配置连接

1. **在"计算机"字段输入:**
   ```
   LA095.macincloud.com:6000
   ```
   或使用 IP:
   ```
   74.80.242.95:6000
   ```

2. **点击"显示选项"**（展开更多设置）

3. **在"常规"标签页:**
   - 计算机：`LA095.macincloud.com:6000`
   - 用户名：`user285049`
   - 勾选"允许我保存凭据"

4. **在"显示"标签页:**
   - 选择合适的分辨率（推荐：1920x1080）
   - 或选择"全屏"

5. **在"本地资源"标签页:**
   - 勾选"剪贴板"（方便复制粘贴）
   - 勾选"驱动器"（可选，用于文件传输）

6. **点击"连接"**

## ⚠️ 常见问题解决

### 问题 1：无法连接到远程计算机

**解决方案：**
1. 检查网络连接
2. 确保端口 6000 未被防火墙阻止
3. 尝试使用 IP 地址而不是域名
4. 检查服务器是否在线（登录用户门户查看）

### 问题 2：密码错误

**解决方案：**
1. 确认密码：`kss66081hjv`（注意大小写）
2. 尝试复制粘贴密码（避免输入错误）
3. 清除保存的凭据后重试：
   - 打开"凭据管理器"
   - 删除旧的 MacinCloud 凭据
   - 重新连接

### 问题 3：连接很慢

**解决方案：**
1. 降低显示质量：
   - 在连接设置中选择较低的颜色深度
   - 禁用桌面背景和主题
2. 使用低图形连接文件（从下载的 ZIP 中）

### 问题 4：需要输入域名

**解决方案：**
- 用户名格式尝试：
  - `user285049`
  - `macincloud\user285049`
  - `LA095\user285049`
